**UAS Machine Learning**  
**Prediksi Timbulan Sampah dan Prioritas Penanganan Kecamatan Kota Tasikmalaya**

Repository ini merupakan proyek **Ujian Akhir Semester (UAS)** mata kuliah **Machine Learning**. Proyek ini bertujuan untuk memprediksi jumlah timbulan sampah pada setiap kecamatan di Kota Tasikmalaya serta menentukan prioritas penanganan sampah berdasarkan data historis menggunakan metode Linear Regression.

Peningkatan jumlah penduduk dan aktivitas masyarakat menyebabkan timbulan sampah di Kota Tasikmalaya terus meningkat dan berbeda pada setiap kecamatan. Data timbulan sampah yang tersedia masih bersifat historis dan belum dimanfaatkan secara optimal untuk mendukung pengambilan keputusan. Melalui proyek ini, pendekatan machine learning digunakan untuk menghasilkan prediksi timbulan sampah dan rekomendasi prioritas penanganan secara objektif dan terukur.

**Tujuan Proyek:**
- Menerapkan metode **Linear Regression** pada studi kasus nyata  
- Memprediksi jumlah timbulan sampah per kecamatan  
- Menentukan prioritas penanganan sampah berdasarkan hasil prediksi
